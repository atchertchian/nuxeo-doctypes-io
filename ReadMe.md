# nuxeo-doctypes-io

This module exposes a REST API to fetch JSON definition of Document types and schemas.

    http://127.0.0.1:8080/nuxeo/site/types

Exports all declared document types and associated schemas.

    http://127.0.0.1:8080/nuxeo/site/types?allSchemas=true

Exports all declared schemas and all declated document types.
